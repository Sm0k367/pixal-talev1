# PixelTale - AI Image to Story Generator

Transform photos into immersive AI-narrated stories with 8 unique genres.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## 🎮 How It Works

1. Upload an image (drag & drop)
2. Pick a story genre
3. Click "Generate Story"
4. Listen to AI narration

## 📋 Genres

- Cinematic
- Fantasy
- Mystery
- Romance
- Horror
- Adventure
- Bedtime Story
- Travelogue

## 🔧 Tech Stack

- React 18 + Vite
- Vercel Serverless Functions
- Groq API (LLaMA Vision)
- Web Speech API

## 📦 Deployment

```bash
npm run build
git push
```

Deploy to Vercel → Add `GROQ_API_KEY` env var → Done!
